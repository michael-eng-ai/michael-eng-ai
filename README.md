# Olá, eu sou o Michael Barbosa 👋

![Profile Banner](https://capsule-render.vercel.app/api?type=waving&color=gradient&height=200&section=header&text=Senior%20Data%20Engineer&fontSize=40&fontAlignY=38&desc=Transforming%20raw%20data%20into%20actionable%20insights&descAlignY=55&descAlign=50)

Sou um **Engenheiro de Dados Sênior** com forte paixão por desenhar, construir e otimizar plataformas de Big Data modulares e escaláveis. Minha missão é ajudar as empresas a extrairem o máximo de valor estratégico dos seus dados, usando tanto as melhores práticas de Engenharia de Dados quanto integrando as mais recentes inovações em Inteligência Artificial Generativa (GenAI).

Com passagem por empresas como **Five Acts**, **Act Digital**, **Vert** e **ART IT**, tenho consolidado meu foco em migrações para Cloud (AWS, Azure, GCP), processos de ETL/ELT distribuídos com ferramentas modernas (Databricks, DBT, Apache Airflow) e streaming em tempo real (Kafka).

## 🏆 Projetos em Destaque no Portfólio

Aqui estão alguns projetos que desenvolvi mostrando meu conhecimento técnico aplicado a problemas de negócios reais:

### 🤖 1. [AI-Data-Analyst-Bot](https://github.com/michael-eng-ai/AI-Data-Analyst-Bot) 🌟 *NOVO!*
*Um assistente virtual híbrido utilizando arquiteturas de GenAI para análise de dados e leitura de documentos.*
- **Arquitetura:** `Text-to-SQL` (Consultas diretas a relatórios de banco de dados) + `RAG` (Busca semântica em documentos de negócio).
- **Tech Stack:** Python, Streamlit, Langchain, Google Gemini 1.5, ChromaDB (Vector Search), SQLite, Docker, GitHub Actions (CI/CD) e Terraform (GCP).

### 🏗️ 2. [AWS-BigData](https://github.com/michael-eng-ai/AWS-BigData)
*Implementação de um Data Lake escalável na AWS para processamento e exploração de dados massivos.*
- **Tech Stack:** AWS S3, AWS Glue, Amazon Athena, crawlers e ETL com Python.

### 🏭 3. [abinbev_case](https://github.com/michael-eng-ai/abinbev_case)

### 🔄 4. [airbyte-dbt](https://github.com/michael-eng-ai/airbyte-dbt)

- **Tech Stack:** Airbyte (Data Integration), DBT (Data Build Tool), PostgreSQL.
